# knostr (Swift Package)

knostr は Kotlin Multiplatform 向けの Nostr プロトコルライブラリです。これは iOS と macOS 向けの Swift Package Manager ディストリビューションです。

## インストール

`Package.swift` に以下を追加してください:

```swift
dependencies: [
    .package(url: "https://github.com/uakihir0/knostr-spm.git", from: "0.0.1")
]
```
