# knostr (Swift Package)

knostr is a Nostr protocol library for Kotlin Multiplatform. This is the Swift Package Manager distribution for iOS and macOS.

## Installation

Add the following to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/uakihir0/knostr-spm.git", from: "0.0.1")
]
```
